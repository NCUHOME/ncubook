# 南昌大学官网信息抓取与内容补充记录

日期：2026-04-25  
用途：为“此间 / 小家园”知识库补充官方来源，提高回答可追溯性，并为后续 Agent 评估提供标准来源。

## 抓取原则

- 优先使用南昌大学官方域名与职能部门页面：`ncu.edu.cn`、`jwc.ncu.edu.cn`、`xgc.ncu.edu.cn`、`xszz.ncu.edu.cn`、`xxwl.ncu.edu.cn`、`net.ncu.edu.cn`、`hqbzc.ncu.edu.cn`。
- 不直接把官网全文复制进手册，只提取学生需要理解的办理路径、适用范围、时间节点、风险提醒和联系方式。
- 涉及资格、金额、时间、名额、收费、办理地点的信息，必须保留来源链接和更新时间。
- 商业套餐、学院细则、窗口服务时间等高变化信息标记为“待核实”。

## 对 Agent 的价值

- 可作为 RAG 的“标准来源白名单”。
- 可用于 Eval Set：要求涉及政策、名额、费用、资格的问题必须引用官方来源或标记“需核实”。
- 可用于 Trust Boundary：涉及奖学金、转专业、资助、收费、报修费用等问题，Agent 不应给绝对结论，应提醒以当年通知和学院/窗口确认为准。
