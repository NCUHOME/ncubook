# 南大家园

## 南大家园能帮你做什么

一款专属于南昌大学学子的校园服务类 APP，智慧互联的校园，从这里开始。

- 📚 查课表：不用翻纸质课表！实时显示上课教室、任课老师、课程时间！
- 📍 校园地图：前湖校区太大找不到教学楼？输入 “建工楼”、“音乐厅” 直接导航。还可以查看每一栋楼的详细介绍。
- ⚡️ 电费查询：宿舍还剩多少电费？打开 App 就能查电费余额，同时监控每天的用电量，轻松管理用电习惯。
- 🍜 美食指南：各校区食堂推荐菜、营业时间、是否需要排队，甚至周边好吃的小店都能找到。
- 💡 还有更多有趣的功能等你发掘~

## **如何下载？**

### **方式一：网页直达下载**

用手机浏览器打开链接：https://incu.ncu.edu.cn 选择对应系统下载。在微信/QQ 中请点击右上角按钮在浏览器中打开，避免应用跳转限制。

:::tip[提示]

华为用户遇到安装风险提示可断开网络重新安装，鸿蒙 Next 用户可下载卓易通浏览器后打开链接安装。

:::

### 方式二：官网入口查找

1. 复制官网链接到浏览器：https://home.ncu.edu.cn
2. 点击页面顶部的【产品】栏目
3. 找到【南大家园】，按使用的设备选择下载方式（苹果用户点击 “App Store”，安卓用户点击 “Android 下载”）

# **如何注册？简单快捷！**

1. 打开微信小程序【南昌大学招生服务】，点击录取查询获得学号
2. 打开南大家园，填写学号等相关信息进行账号绑定

**温馨提示：** 请确保下载最新版本南大家园APP，注册成功后，使用手机号 + 验证码即可轻松登录！也可参考下图完成。
