# 辅修&第二学士学位

## 辅修

### 报名资格与条件

- **对象**：南昌大学**全日制普通本科一、二年级学生（不含第二学士学位学生）** 可以申请辅修专业/辅修学士学位。
- **品行**：在校期间未受过纪律处分，且政治思想表现良好。
- **学业**：已修课程的**平均学分绩点在 2.0 以上**，或者对申报专业有特长和兴趣，并获得承办学院的审核同意。
- **数量限制**：每位学生在校期间最多只能修读一个辅修专业/辅修学士学位。

### 费用与管理

- **报名时间**：学校每年**春季学期末**会公布招生通知并接受学生报名。
- **收费标准**（按学年收费，一次性缴清）：
  - 文科（文、史、哲）类：3500 元/学年
  - 文科（经、管、法、外）类：3800 元/学年
  - 理工科（I）类：4000 元/学年
  - 理工科（II）类：4200 元/学年
- **退费**：如果中途退出，已交的费用不予退还。

### 教学与成绩

- **授课方式**：辅修课程可能跟普通专业班一起学习，或单独开设辅修专业班。单独开设的辅修专业班原则上会在双休日上课。
- **课程要求**：辅修专业课程的教学和质量要求与主修该专业的标准相同。
- **成绩计算**：辅修课程的成绩会单独记录为"辅修"性质，并可以单独打印成绩单。这些成绩不计入主修专业的平均学分绩点排名。
- **免修**：如果学生在主修专业已修过课程，且课程规格和学分不低于辅修专业的要求，可以申请免修。

### 毕业与证书

- 在规定的学习年限内，如果学生获得了主修专业的学士学位，并且修满了辅修专业规定的全部学分，将获得**南昌大学辅修专业证书**。
- 如果同时符合学校学士学位授予的规定，在主修学士学位证书上会注明辅修学士学位。

:::tip
详细官方通知请见：[关于做好2025年辅修专业/辅修学士学位招生工作的通知](https://jwc.ncu.edu.cn/jwtz/09e20f41f1754c199c9c40532643da81.htm)
:::

## 第二学士学位

### 招生对象

主要面向**普通高校本科毕业，并获得学士学位的应届毕业生，以及往届本科毕业生（含在职人员）**。

### 年龄限制

2024 年的招生简章要求年龄一般不超过 35 周岁。

### 专业限制

报考第二学士学位的专业应与你原主修的本科专业分属不同学科门类。例如，如果你主修的是工学，你可以报考法学或管理学类的第二学士学位。

### 学习年限与学籍管理

- **学制**：两年
- **学籍**：学生在校期间的学籍管理参照本科生的相关规定执行
- **课程**：课程设置以专业核心课程为主，学生可以申请免修部分与原专业相似的公共基础课

### 毕业与学位授予

- **毕业条件**：在规定的学习年限内，修满教学计划规定的所有学分，完成毕业论文或毕业设计，并通过答辩
- **学位证书**：符合条件的毕业生将同时获得毕业证书和学士学位证书，证书上会注明第二学士学位的字样
- **就业**：毕业时享受与当年应届本科毕业生同等的待遇

详细官方通知请见：[南昌大学2025年第二学士学位招生简章](https://zjc.ncu.edu.cn/zs/f/newsCenter/article/152eb901866244fcb1d6078948bb7f8f)

## 辅修 vs 第二学士学位 区别

| 特性 | 辅修学士学位 | 第二学士学位 |
| --- | --- | --- |
| **就读阶段** | 本科在读期间，同时修读两个专业 | 获得第一个学士学位后，作为新的学历教育修读 |
| **招生对象** | 本校在读本科生（大一、大二） | 已获得学士学位的本科毕业生 |
| **毕业证书** | 只颁发一个主修专业的毕业证书 | 颁发独立的毕业证书和学士学位证书 |
| **学位证书** | 主修学士学位证书上注明辅修学士学位 | 颁发独立的第二学士学位证书 |
| **学籍管理** | 保留主修专业学籍 | 属于新的学历层次，学籍独立 |
| **学习年限** | 不延长本科学习年限 | 学制两年 |
| **毕业待遇** | 辅修经历体现在成绩单和毕业证书上 | 享受与当年应届本科毕业生同等的就业待遇 |
