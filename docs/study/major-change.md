# 不喜欢本专业/想学其他专业

如果你发现自己不喜欢当前的专业，或者想学习其他专业的知识，不要担心，你有多种选择。

## 🔄 转专业

### 转专业政策
- 通常在大一第二学期或大二第一学期可以申请转专业
- 需要满足一定的成绩要求（通常要求绩点在年级前列）
- 不同学院、不同专业的转入要求不同

### 转专业流程
1. 关注教务处发布的转专业通知
2. 了解目标专业的接收条件和名额
3. 准备相关材料（成绩单、申请理由等）
4. 提交申请并参加面试（如需要）
5. 等待审核结果

### 注意事项
- 转专业有名额限制，竞争可能较为激烈
- 部分专业不接受转入或有特殊要求
- 转专业后可能需要补修一些课程
- 艺术类、体育类等特殊专业通常不能转入

## 📚 辅修/第二学士学位

如果不想转专业，但对其他专业感兴趣，可以考虑：
- **辅修**：在完成本专业学习的同时，修读另一个专业的部分核心课程
- **第二学士学位**：修读另一个专业的全部课程，获得第二个学士学位

详见：[辅修&第二学士学位](./double-degree.md)

## 💻 自主学习

### 在线课程平台
- 中国大学MOOC
- 学堂在线
- Coursera
- edX

### 学习资源
- 图书馆电子资源
- 专业相关书籍
- 开源项目和文档
- 技术社区和论坛

### 实践机会
- 参加相关的学生社团
- 参与科研项目
- 参加竞赛
- 寻找实习机会

## 🎯 职业规划建议

### 了解自己
- 分析自己的兴趣、特长和价值观
- 思考不喜欢当前专业的具体原因
- 确定自己真正想学习和从事的方向

### 咨询交流
- 与辅导员、班主任沟通
- 向学长学姐请教
- 咨询职业规划中心
- 与家长、朋友讨论

### 理性决策
- 不要仅凭一时兴趣做决定
- 了解目标专业的学习内容和就业情况
- 评估转专业的成本和收益
- 制定可行的学习计划

## ⚠️ 重要提醒

- 即使不喜欢当前专业，也要认真完成学业，保持良好成绩
- 转专业不是唯一出路，很多人从事的工作与专业并不对口
- 大学期间培养的学习能力、思维方式比具体专业知识更重要
- 无论选择什么方向，持续学习和努力都是成功的关键
