# 班干部

大学里的班干部主要包括班长、副班长、团支书、学习委员、生活委员、体育委员、文艺委员、组织委员、宣传委员和心理委员等。

不同的学校和学院有不同的配置，但是**班长、团支书和学习委员**这三个职位可以说是班委的核心职位，也是与老师最近、需要投入精力最多的职位。

## 与中学阶段的不同

### 当选方式

- 高中班委：可能更多采用班主任或科任老师任命的方式
- 大学班委：通常通过民主选举产生，过程更加公开透明

### 影响力与作用

- 高中班委：辅助教师，主要体现在维护班级纪律和协助教学
- 大学班委：在班级和学院中拥有较大的影响力，不仅管理班级事务，还参与校园文化建设和学生权益保护，是连接学生与学校管理层的重要纽带

### 工作内容

大学班委需要承担更多的责任和义务，如班长需要组织召开班委会和班会、处理班级重大事务等；团支书则需要负责班级团员的团日活动组织等工作。

## 担任班干部能得到什么？

1. **通用问题解决能力**：组织协作、沟通、问题解决、时间管理
2. **评奖评优优先**
3. **加综测分**
4. **信息差**：事先知道学校的一些内部消息，优先获得讲座或活动的名额
5. **拓展人脉圈**：有机会与学院负责老师、优秀学长学姐建立联系，获得竞赛机会
6. **责任感与成就感**

## 需要付出什么？

所有职位都会在刚开学任职后的两个月内有不同程度的忙（会占据更多的休息时间）。随着在校时长增加，会越来越不怎么忙。

各职位工作量：

- **班长**：日常小忙。转发通知，与不同同学交流，偶尔和班干、班级开会，与老师交流得较多。
- **团支书**：日常较忙。参加培训，学习智慧团建，开展团日活动，督促同学完成青年大学习。和导员联系紧密，容易被老师叫去干活。
- **学习委员**：工作量视课程决定。与老师交流得多，某些教师会默认学委做课代表而安排学习任务。
- **生活委员**：工作量较低。转发督促同学填一些表的通知，贫困生、各种助学金、医保等。
- **其他委员**：工作量较低。均为转发通知的职位，偶尔找同学参加一些活动或竞赛。

## 如何竞选班干

### 竞选流程

开学第一周内事情比较多，此时多是辅导员或班主任临时指定或自告奋勇，选定临时班干。这些临时班干在之后的民主选举中往往会有更大的优势。

之后在开学工作相对稳定/军训结束后，班级将举行班会进行正式的班干竞选。想要竞选的同学可以报名并做介绍，然后由同学们民主投票决定最后的人选。

### 竞选稿要点

1. **自我介绍**：姓名、家乡，让班集体对你有初步印象
2. **工作理解**：解释你竞选的职位在班级中的作用
3. **竞选原因**：分享相关经验和个人优势
4. **计划行动**：阐述当选后将如何开展工作
5. **拉票呼吁**：认可其他竞选者，礼貌请求投票
