# 辅修与第二学士学位

<p className="doc-meta">页面更新：2026-04-26 · 来源：<a href="https://jwc.ncu.edu.cn/">南昌大学教务处</a>、<a href="https://zjc.ncu.edu.cn/zs/f/newsCenter/article/152eb901866244fcb1d6078948bb7f8f">2025年第二学士学位招生简章</a> · 待核实：辅修当年通知</p>

## 辅修适合谁

辅修/辅修学士学位更适合仍在本科阶段、想在主修专业之外系统学习另一个方向的同学。

你需要重点确认：

- 当年是否开放你想读的辅修专业。
- 面向哪些年级和学生类型。
- 是否有绩点、纪律处分、课程基础或学院审核要求。
- 收费标准、上课时间和是否会影响主修课程。
- 退出、免修、成绩记录和证书规则。

:::warning
不要只按往年经验报名。辅修招生计划、收费和条件都可能变化，以当年教务处通知为准。
:::

## 第二学士学位适合谁

第二学士学位通常面向已经获得学士学位的本科毕业生，属于毕业后再修读一个本科层次专业。它和“在校期间同时修一个辅修”不是一回事。

根据 2025 年招生简章，报考对象、专业、报名方式、录取规则、学制和培养方式都以招生简章为准。具体年龄、专业门类限制、材料要求等不要套用往年版本。

## 两者区别

| 维度 | 辅修 / 辅修学士学位 | 第二学士学位 |
| --- | --- | --- |
| 就读阶段 | 本科在读期间 | 已获得学士学位后 |
| 学籍关系 | 保留主修专业学籍 | 新的学历教育阶段 |
| 时间投入 | 通常与主修并行 | 通常单独修读 |
| 来源入口 | 教务处通知 | 招生与就业工作处招生简章 |
| 风险点 | 时间冲突、费用、退出规则、证书规则 | 报考资格、专业限制、录取规则、就业安排 |

## 决策建议

- 如果只是对某个方向感兴趣，先通过选修课、公开课、社团、项目和实习验证兴趣。
- 如果目标是就业转向，辅修证书未必比作品、项目、实习更关键。
- 如果目标是升学或跨专业申请，重点看目标院校/导师是否认可相关课程和经历。
- 报名前算清楚时间成本，尤其是周末上课、考试冲突和主修绩点压力。
