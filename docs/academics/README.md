# 学业

这里整理大学四年里最常被反复查询的学业流程：绩点、课程、考试、转专业、双学位、考勤、英语和体育。每一页都尽量把“适用对象、办理路径、注意事项”拆开写。

## 常见问题

- [学分绩点](/docs/academics/credits-gpa)：成绩、绩点、学分之间的关系。
- [教务办事入口](/docs/academics/academic-services)：选课、考试、成绩、学籍学位等官方入口。
- [培养方案与课程](/docs/academics/curriculum)：怎么看自己的培养方案，哪些课必须修。
- [考试与缓补考](/docs/academics/exams)：考试安排、缓考、补考和成绩相关事项。
- [转专业](/docs/academics/major-change)：申请条件、时间节点、材料和风险。
- [辅修与第二学士学位](/docs/academics/double-degree)：适合谁、怎么申请、压力在哪里。

## 使用建议

学业规则经常和学院、年级、培养方案有关。看到关键结论时，请注意页面里的更新时间和来源；不确定时，可以让小家园继续追问“这条规则适用于我这一届吗”。
