# 生活

这里整理校园生活里的高频办事和日常问题：吃饭、交通、报修、电话、家园账号和常用软件。它更像一张可检索的生活地图，帮你少问几次“这个要去哪办”。

## 常用入口

- [饮食](/docs/campus-life/dining)：校内餐饮区域和日常选择。
- [校内交通](/docs/campus-life/campus-transport)：校区内出行方式和路线信息。
- [校外交通](/docs/campus-life/external-transport)：去车站、机场和市区的交通选择。
- [报修](/docs/campus-life/repair)：宿舍、设施和公共区域问题怎么提交。
- [常用电话](/docs/campus-life/phone-directory)：遇到问题时先找谁。
- [医保 / 校医院](/docs/campus-life/medical-insurance)：医保办、校医院急诊和就医报销咨询入口。
- [图书馆](/docs/campus-life/library-services)：图书馆官网、借阅和电子资源入口。
- [家园账号注册](/docs/campus-life/jiayuan-register)：家园相关服务的账号与入口。

## 信息反馈

生活类信息变化很快。如果你发现地点、窗口、电话或流程过期，可以在页面底部反馈，也可以去共建页提交补白。
