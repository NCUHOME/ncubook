# 共建

此间的准确性来自持续校订。你可以提交新信息，也可以指出旧信息哪里不准确。我们更关心“信息能不能被后来的人继续使用”，而不是一次性写得多完整。

## 你可以贡献什么

- 信息过期：地点、电话、窗口、时间、流程发生变化。
- 信息缺口：小家园答不上来，或文档里完全没有覆盖。
- 经验补充：某个流程的真实办理路径、常见坑点、需要准备的材料。
- 来源校订：补充官方通知、页面截图、公告链接或办理入口。

## 推荐格式

请尽量说明：问题是什么、你看到的来源是什么、这条信息适用于哪个校区/学院/时间段、是否需要再次核实。
