# 关于此间

import QuoteCard from '@site/src/hight_light/QuoteCard';

<QuoteCard icon="源">
此间不是官方通知站，也不是聊天记录合集。它更像一个校园知识入口：把真实经验、公开信息和同学补白整理到一起，让后来的人少走一点弯路。
</QuoteCard>

## 为什么要做

大学里的很多信息并不是不存在，而是分散在不同地方：学院通知、群聊截图、学长学姐经验、公众号推文、办事窗口和同学口耳相传。新同学第一次遇到这些问题时，往往不知道该相信谁，也不知道应该去哪确认。

此间希望降低这种信息差。我们把常见问题整理成文档，也提供小家园问答入口；能确定的内容写成指南，暂时不确定的内容标注状态，仍在变化的信息放进“此刻”继续观察。

## 此间和小家园的关系

- 此间是学生端的信息入口，负责承载稳定、可追溯的校园指南。
- 小家园是问答助手，负责帮你从知识库里找答案，并把答不上来的问题暴露出来。
- 校订与补白是维护机制，负责把过期信息、缺口和新经验重新回流到文档。

## 我们如何判断一条信息是否可靠

一条信息越重要，就越需要补齐来源、更新时间和适用范围。比如涉及流程、费用、地点、材料和截止时间的信息，最好能对应到官方通知、页面截图、现场办理经验或多位同学反馈。

如果信息尚未稳定，我们不会把它包装成确定答案，而是标注“待核实”“补白中”或“已回流”。这也是此间区别于普通聊天机器人的地方：它不仅回答问题，也记录答案是怎么来的。
