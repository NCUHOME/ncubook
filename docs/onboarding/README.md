# 入学指南

这里整理新生入学前后最容易卡住的问题：报到准备、宿舍生活、校园卡、学生证和网络。目标不是把所有通知重新复制一遍，而是把“第一周真的会用到”的步骤讲清楚。

## 推荐先看

- [新生报到指南](/docs/onboarding/freshmen-guide)：确认时间、地点、材料和报到动线。
- [生活必需品](/docs/onboarding/essentials)：区分必须带、到校买和可以以后再补的东西。
- [宿舍生活](/docs/onboarding/dorm-life)：提前了解住宿环境和常见注意事项。
- [校园卡与校园一码通](/docs/onboarding/campus-card)：领取校园码、使用场景和卡务提醒。
- [校园网与校园电话卡](/docs/onboarding/network)：设备连接、账号登录、报修和套餐风险。

## 如果你不知道该看哪篇

可以直接问小家园：“我明天到校报到，今天还要准备什么？” 它会优先从这些指南里找答案，并告诉你信息来源。
