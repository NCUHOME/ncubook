# 必备物品

## 证件、材料

- **身份证**（最好也准备正反面的清晰照片电子版）
- **证件照**
  - 一寸、两寸最好都要
  - 保存好电子版，后续有电子上传用途，也方便以后打印照片
  - 标准电子版像素为 `640x480`，可自行保存高清版本，但可能因大小超出限制而不可上传
- **录取通知书**
- **学籍档案**（大部分学校都会自行邮寄，少数地区要求学生自带）

## 生活用品

### 日常用品

- 洗漱用具
- 盆、衣物桶
- 指甲刀、掏耳勺
- 水杯（不需要热水壶，宿舍有饮水机）
- 卫生纸
- 防晒霜（军训记得多涂！涂满！）
- 剪刀、裁纸刀
- 花露水、驱蚊器
- 驱虫剂

### 药品

- 藿香正气
- 感冒药
- 蒙脱石散
- 云南白药
- 蚊虫叮咬药

## 电子设备

:::warning
宿舍限 800W 功率，携带的时候记得检查自己的设备功率。
:::

- **插座**：请购买安全的品牌，网购！这是寝室用电安全的最关键一步。
- **台灯**：宿舍有大灯，但是坐在桌前台灯会照得更清楚，请注意护眼问题，务必购买低频闪型护眼台灯。

## 学习用品

- **教材**
  - 学校有二手交易群
  - 新生大一整个学年的教材在开学学校已经统一订购，大二后有两个选择：自行购买或者跟随学校订购
- **文具**
  - 中性笔
  - 2B 铅笔
  - 橡皮
  - 计算器
