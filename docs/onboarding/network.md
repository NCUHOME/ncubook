# 校园网与校园电话卡

## 校园网怎么连

网络与信息中心说明中提到：

- 教学、办公、科研区域无线信号通常为 **NCUWLAN**。
- 学生宿舍区域无线信号通常为 **NCU2.4G** 或 **5G**。
- 连接后需要通过认证页面登录。

如果浏览器没有自动弹出认证页面，可以手动打开：

- 教学/办公/科研区域：`http://aaa.ncu.edu.cn` 或 `222.204.3.221`
- 学生宿舍区域：`http://3a.ncu.edu.cn` 或 `222.204.3.154`

忘记上网密码时，可进入“我的南大”融合门户 `my.ncu.edu.cn`，在首页点击“忘记密码”，通过手机验证码修改。

## 立即办理 / 报修

如果校园网连不上、认证失败、宿舍网络异常，可以优先走官方报修入口：

- 线上报修：进入[校园网报修服务](https://xxwl.ncu.edu.cn/info/1010/6179.htm)，点击页面中的“立即办理”。
- 线下报修：前湖校区天健园五栋 5107 室，网络与信息中心前湖校区现场服务点。
- 咨询电话：**0791-83969312**。

官网页面标注的服务时间：

| 类型 | 时间 |
| --- | --- |
| 线上报修 | 周一至周日 8:00-22:00 |
| 线下报修 | 周一至周五 9:00-11:30、14:00-16:30 |

节假日、寒暑假或系统维护期间，服务时间可能临时调整，以网络与信息中心通知为准。

## 校园电话卡不是校园卡

校园电话卡是运营商销售的手机卡或宽带套餐，不是学校统一发放的实体校园卡，也不是校园一码通。

办理前至少确认：

- 月租、优惠期、合约期和到期后的资费。
- 是否有违约金、注销限制或必须使用满多久。
- 宽带/校园网账号是否与手机号绑定。
- 同时在线设备数量。
- 售后地点和故障处理方式。

:::warning
不要只听口头承诺。涉及费用和合约的内容，以运营商合同、营业厅书面说明和当年套餐规则为准。
:::
