# 图标语义表

图标库：Lucide React。除品牌标志外，禁止使用其他图标库、emoji 或手写 SVG 作为功能图标。

## 全局规则

- 默认使用 20px；导航或悬浮主操作可使用 24px。
- 使用固定线宽，最终数值在第一个视觉样张中冻结。
- 图标按钮必须有文字可见标签、`aria-label` 或两者兼具。
- 不以图标单独表达高风险或难懂操作。

## 固定映射

| 语义 | Lucide 图标 | 使用位置 |
|---|---|---|
| 打开页面树 | `Menu` | 文档页左上 |
| 返回上级 | `ArrowLeft` | 文档页左上 |
| 关键词搜索 | `Search` | 文档页右上、搜索页 |
| 打开 AI 提问 | `Sparkles` | 右下悬浮入口 |
| 发送问题 | `ArrowUp` | AI 输入框 |
| 关闭抽屉/弹层 | `X` | 抽屉与弹层 |
| 展开下级页面 | `ChevronRight` | 页面树 |
| 收起层级 | `ChevronDown` | 页面树、折叠内容 |
| 外部链接 | `ArrowUpRight` | 外链标识 |
| 返回回答 | `ArrowLeft` | 从来源页返回回答 |
| 复制链接 | `Link` | 分享/复制链接 |
| 文件附件 | `Paperclip` | 文档附件 |
| 信息提示 | `Info` | 中性说明 |
| 风险提醒 | `TriangleAlert` | 需核实或风险说明 |
