# 组件契约

状态：首版。新增 UI 组件前必须更新本文件并添加可隔离审阅样例。

| 组件 | 职责 | 必须行为 | 禁止行为 |
|---|---|---|---|
| `AppHeader` | 提供当前页面的最少导航 | 返回、页面树、关键词搜索；保持 44px 触控区 | 常驻占用多行、塞入无关入口 |
| `PageTreeDrawer` | 浏览当前板块页面树 | 焦点锁定、关闭后返回触发元素、支持上级返回 | 作为全站无限侧栏、遮住正文后无法退出 |
| `SearchField` | 输入关键词或问题 | 清除、提交、加载、空状态与可见标签 | 混用关键词搜索与 AI 问答结果 |
| `SearchResultItem` | 展示全文检索命中 | 标题、板块、段落摘录、稳定锚点链接 | 生成摘要答案或加入无关推荐 |
| `FloatingAskButton` | 打开页面上下文 AI | 文档页携带 page 与 heading context；可访问标签；避开 safe area、键盘和正文附件 | 作为常驻信息面板；覆盖正文关键控件或遮挡手机系统区域 |
| `AskSheet` | 承载 AI 问答 | 显示当前上下文、来源、跳转与返回回答 | 人格化寒暄、隐藏不确定性 |
| `ArticleRenderer` | 渲染站内富文档 | 保留块树、锚点、表格、多栏和附件语义 | 将长文改造成卡片流或丢弃结构 |
| `CalloutBlock` | 呈现提醒语义 | 以低干扰视觉区分信息/提醒/风险 | 用作营销横幅或替代正文层级 |
| `TableBlock` | 呈现表格 | 手机横向滚动、固定/清晰表头、可读内容 | 删除列、把数据改写为图形卡片 |
| `ColumnsBlock` | 呈现并列内容 | 宽屏并列、手机顺序堆叠 | 在窄屏保留不可读的双列 |
