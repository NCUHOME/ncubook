## 手册使用指南

- 手册分为两个部分，正文内容部分和大模型（小家园）智能提问部分。
- 鉴于手册内容较多，可以先粗略浏览一遍所有内容，方便以后需要的时候可以快速找到想要的资料
- **觉得内容写得有点复杂？**可以试试点击右下角的小家园对手册进行提问，小家园会耐心解答~
- 发现没有找到自己想要的内容？先试试向小家园提问，若还是没有找到答案，请在对应页面最下方点击**差评**，会有一个表单让您填写。
- 本手册目前还处于萌芽阶段，目标是成为一个覆盖南昌大学生活方方面面的资源汇合，**欢迎大家投稿联系~**

## 为什么会去做这个手册

在高考完毕业的那个暑假，一直被一股未知的焦虑捆绑，也在互联网上刷了太多关于大学的帖子。说大学是小社会，又说大学是象牙塔。很多完全没有接触的概念扑面而来：学分、绩点、选修课等等。对于这种未知的恐惧让我迫切想要去找到一个可以给我提供一个大学解读的“教材”。

于是这样一本书出现在了我的视野——《金榜题名之后：大学生出路分化之谜》，阅读时感叹良多，文中对于大学诸多详实的例子，极大地帮助我快速了解大学四年的全部面貌。

后面又发现了[上海交通大学生存手册](https://survivesjtu.gitbook.io/survivesjtumanual/)。阅读完全文后感慨良多，激发我开始对国内教育体系的思考，以及对于大学的真正面貌揭开后的无奈。在这里非常感谢所有这些撰写学生手册的前辈们，正是他们探索付出所带领的潮头，才有了后续这些如雨后春笋冒出的各种生存手册

> 这是电子书，推荐导入微信读书进行阅读

在大学的一年生活中，由于刚进来仍然保持着高中的学生思维（简称“太想进步了”），获取了一点资源，结识了一点朋友。每次和学长学姐的闲聊，或者机缘巧合获取了一些信息的时候，总忍不住感叹大学那坚厚的信息壁垒。

在看到那么多优秀的前辈们做出的生存手册，为了降低大学信息差而甘愿做出奉献。我就想，为什么南昌大学不可以有一个呢？于是就有了这个手册的立项诞生。
