## 怎么使用此间

- 想直接办事：先进入对应栏目，查看已经整理好的指南。
- 不知道去哪找：打开右下角的小家园，直接把问题说出来。
- 发现信息过期：在页面底部反馈，或进入共建页提交补白。
- 看到新的校园变化：放进“此刻”，等信息稳定后再回流到正式指南。

## 信息如何变得可靠

此间不会把所有聊天记录都直接当成答案。每条重要信息都应该尽量补齐来源、更新时间和适用范围。小家园回答不了的问题，会被记录为信息缺口；同学补充过的内容，会逐步回流到正式文档里。
