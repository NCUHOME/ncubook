# 升学与求职

本板块面向开始考虑下一步发展的同学：考研、保研、竞赛、评优、科研和项目经历。它不会替你做决定，但会把常见路径、关键节点和可准备的材料先摆清楚。

## 目前已有内容

- [升学准备](/docs/career/postgraduate)：考研、保研和长期准备节奏。
- [评奖评优](/docs/career/awards)：奖学金、荣誉和材料整理。
- [竞赛与科研](/docs/career/innovation-research)：项目经历、竞赛和科研机会。
- [就业手续与档案](/docs/career/employment-services)：三方协议、就业推荐表、档案去向和校园招聘入口。

## 仍在建设

发展类内容很依赖真实经验。欢迎补充不同学院、不同年级、不同路径的案例；此间会把具体经验整理成更稳定的指南，而不是只留下零散建议。
