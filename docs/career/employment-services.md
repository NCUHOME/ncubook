# 就业手续与档案入口

<p className="doc-meta">页面更新：2026-04-26 · 来源：<a href="https://www.ncu.edu.cn/info/1042/1159.htm">招生与就业工作处服务指南</a>、<a href="https://jy.ncu.edu.cn/">南昌大学就业信息网</a> · 待核实：当年毕业手续和学院通知</p>

## 哪些问题找就业入口

- 三方协议、就业协议、就业推荐表。
- 校园招聘、宣讲会、招聘会。
- 毕业生就业手续、就业信息填报。
- 档案去向查询、档案转递咨询。
- 升学、出国、灵活就业、未就业等毕业去向确认。

## 办理前先确认

| 问题 | 为什么要确认 |
| --- | --- |
| 你的毕业年份 | 不同年份政策和系统入口可能不同 |
| 学院通知 | 学院通常会统一收材料、审核和提醒时间 |
| 单位性质 | 国企、事业单位、民企、升学、出国材料不同 |
| 档案去向 | 影响后续入职、落户、考公考编和人事手续 |

## 使用建议

- 不要只看群里转发的截图，优先看就业工作处官网和学院毕业班通知。
- 涉及签约、解约、档案、户口、报到等事项时，保留邮件、系统截图和纸质材料。
- 签署就业协议前，确认违约责任、试用期、薪资构成、档案接收和户口安排。
